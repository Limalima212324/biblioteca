⚙️ Atributos Comuns

Atributos globais são atributos que podem ser usados em praticamente todas as tags HTML.

id

Define um identificador único para um elemento. Usado para estilização com CSS (#meu-id) e manipulação com JavaScript (document.getElementById('meu-id')).

<p id="paragrafo-unico">Este parágrafo tem um ID.</p>

class

Associa um ou mais nomes de classe a um elemento. Usado para estilizar múltiplos elementos com CSS (.minha-classe).

<p class="destaque importante">Este parágrafo tem duas classes.</p>

style

Aplica estilos CSS diretamente no elemento (CSS inline). Deve ser usado com moderação.

<p style="color: red; font-size: 20px;">Texto estilizado com CSS inline.</p>

title

Fornece informações extras sobre um elemento. O texto geralmente aparece quando o usuário passa o mouse sobre o elemento.

<button title="Clique aqui para salvar o documento">Salvar</button>

lang

Especifica o idioma do conteúdo do elemento. Ajuda os mecanismos de busca e leitores de tela.

<p lang="en">This paragraph is in English.</p>

hidden

Um atributo booleano que esconde o elemento da página. Ele não é renderizado.

<p hidden>Este parágrafo está escondido.</p>

O parágrafo acima está no código, mas não aparece na tela.
