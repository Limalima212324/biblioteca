☑️ Checkbox (<input type="checkbox">)

O checkbox é usado quando o usuário pode selecionar zero, uma ou várias opções de uma lista.

Como Usar

Para agrupar várias opções, use o mesmo atributo name para todos os checkboxes. Cada um deve ter um value único que será enviado ao servidor se a opção for marcada. É uma boa prática envolver o input e seu texto em uma tag <label> para melhorar a acessibilidade e a usabilidade (permite clicar no texto para marcar a caixa).

Exemplo Prático

Quais tecnologias você conhece?

<label>
  <input type="checkbox" name="tecnologias" value="html"> HTML
</label>
<label>
  <input type="checkbox" name="tecnologias" value="css" checked> CSS
</label>
<label>
  <input type="checkbox" name="tecnologias" value="js"> JavaScript
</label>
