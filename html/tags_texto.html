✏️ Tags de Texto

Estas tags são usadas para estruturar e formatar o conteúdo textual de uma página.

Títulos (<h1> a <h6>)

Definem uma hierarquia de títulos, do mais importante (h1) ao menos importante (h6).

Título Nível 1

Título Nível 2

Título Nível 3

Parágrafo (<p>)

Usada para agrupar um bloco de texto em um parágrafo.

Este é um parágrafo de exemplo. Ele contém várias frases para demonstrar como o texto é agrupado.

Negrito (<strong> e <b>)

Ambas deixam o texto em negrito, mas <strong> indica que o texto tem grande importância (semântica), enquanto <b> é apenas para destaque visual.

Este texto é normal, mas esta parte é muito importante.

Este texto é normal, e esta parte está apenas em negrito.

Itálico (<em> e <i>)

Ambas deixam o texto em itálico. <em> (emphasis) dá ênfase a uma palavra ou trecho. <i> é usado para termos técnicos, nomes estrangeiros ou destaque visual.

Você precisa ver isso! (Ênfase)

O termo de facto vem do Latim. (Termo estrangeiro)

Span (<span>)

É uma tag "coringa" para agrupar pequenos trechos de texto, geralmente para aplicar um estilo CSS específico sem quebrar a linha.

A maior parte do texto é preta, mas esta palavra é vermelha.

Quebra de Linha (<br>) e Linha Horizontal (<hr>)

<br> força uma quebra de linha. <hr> cria uma linha horizontal para separar conteúdos.

Este texto está na primeira linha.
Este está na segunda.

Este texto está após a linha horizontal.

Texto Pré-Formatado (<pre> e <code>)

A tag <pre> preserva os espaços em branco e as quebras de linha do texto. É ideal para exibir blocos de código, geralmente envolvendo a tag <code>.

function helloWorld() {
  console.log("Olá, Mundo!");
}
